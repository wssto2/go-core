Using sqlc alongside GORM

This repository includes a small example showing how to use sqlc-style
generated queries (database/sql) together with GORM for writes.

See: internal/domain/product/sqlc_example.go

What it demonstrates:

- Creating a record with GORM (useful for ORM-based writes)
- Reading the same record with a sqlc-generated queries struct using the
  same underlying *sql.DB (reads are fast and explicit)
- How to read inside a transaction by pulling *sql.Tx from the Transactor
  context (database.SQLTxFromContext) and constructing a tx-bound queries
  instance.

Notes:
- In real projects the Queries struct would be generated by sqlc and live in
  its own package. The example shows the wiring pattern and a tiny stub to
  illustrate the integration.
